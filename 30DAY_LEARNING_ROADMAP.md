# 30-DAY LEARNING ROADMAP
## From Security Ops to Data Analyst / AI SOC Analyst

**Goal:** Job-ready entry-level Data Analyst or AI SOC Analyst

---

## WEEK 1: PYTHON FUNDAMENTALS
### Duration: 5-6 days | Effort: 2-3 hours/day

#### Core Topics
- **Variables & Data Types** (Day 1-2)
  - Strings, integers, floats, booleans
  - Lists, tuples, dictionaries
  - Variable naming conventions
  
- **Control Flow** (Day 2-3)
  - If/else statements
  - Loops (for, while)
  - Break, continue
  
- **Functions** (Day 3-4)
  - Defining and calling functions
  - Parameters and return values
  - Local vs global scope
  
- **File I/O & Error Handling** (Day 4-5)
  - Reading/writing files
  - Try-except blocks
  - Common exceptions

#### Practical Exercise
**Build:** Simple log parser script
- Read a CSV file of security logs
- Filter logs by severity level
- Print filtered results
- Save to new file

**Time:** 1-2 hours
**Tools:** Python, VS Code, Jupyter Notebook

---

## WEEK 2: DATA ANALYSIS WITH PANDAS & NUMPY
### Duration: 5-6 days | Effort: 2-3 hours/day

#### Core Topics
- **Pandas DataFrames** (Day 1-2)
  - Creating DataFrames
  - Reading CSV/JSON files
  - Selecting rows/columns
  - Filtering data
  
- **Data Cleaning** (Day 2-3)
  - Handling missing values
  - Removing duplicates
  - Data type conversion
  - Renaming columns
  
- **NumPy Basics** (Day 3-4)
  - Arrays vs lists
  - Element-wise operations
  - Statistical functions (mean, std, etc.)
  
- **Basic Statistics** (Day 4-5)
  - Descriptive statistics
  - Groupby operations
  - Aggregation functions

#### Practical Exercise
**Build:** Security log analysis dashboard script
- Load 50K+ rows of login logs
- Clean and prepare data
- Calculate statistics (login frequency, unique IPs, times)
- Identify unusual patterns (off-hours logins, multiple IPs)
- Generate summary report

**Time:** 2-3 hours
**Tools:** Python, Pandas, Jupyter Notebook

---

## WEEK 3: MACHINE LEARNING FUNDAMENTALS
### Duration: 5-6 days | Effort: 2-3 hours/day

#### Core Topics
- **ML Concepts** (Day 1-2)
  - Supervised vs unsupervised learning
  - Training vs test data
  - Overfitting and underfitting
  - Model evaluation metrics
  
- **Scikit-learn Basics** (Day 2-3)
  - Installation and setup
  - Preprocessing data
  - Train-test split
  - Scaling features
  
- **Unsupervised Learning** (Day 3-4)
  - K-means clustering
  - Isolation Forest (anomaly detection)
  - DBSCAN
  
- **Classification Basics** (Day 4-5)
  - Logistic Regression
  - Decision Trees
  - Random Forest
  - Model accuracy and confusion matrix

#### Practical Exercise
**Build:** Login Anomaly Detection System
- Use Isolation Forest to detect outliers in login patterns
- Features: login frequency, unique IPs, time of day, location
- Train on 6 months of normal logins
- Test on new data with injected anomalies
- Achieve 90%+ detection rate
- Generate anomaly report

**Time:** 3-4 hours
**Tools:** Python, Pandas, Scikit-learn, Jupyter Notebook

---

- Highlight learning mindset and growth
